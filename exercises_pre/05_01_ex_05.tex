{$f(x) = 4x+5$, $g(x) = \sqrt{x}$}
{ For $f(x) = 4x+5$ and $g(x) = \sqrt{x}$,
\begin{multicols}{2}

\begin{itemize}

\item  $(g\circ f)(0) = \sqrt{5}$

\item  $(f\circ g)(-1)$ is not real

\item  $(f \circ f)(2) = 57$

\item  $(g\circ f)(-3)$ is not real

\item  $(f\circ g)\left(\frac{1}{2}\right) = 5+2\sqrt{2}$

\item  $(f \circ f)(-2) = -7$

\end{itemize}

\end{multicols}}
