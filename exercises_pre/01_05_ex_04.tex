{$f(x) = 2x^3$ and $g(x) = -x^2-2x-3$}
{For $f(x) = 2x^3$ and  $g(x) = -x^2-2x-3$

\begin{multicols}{2}
\begin{itemize}
\item  $(f+g)(2) = 5$
\item  $(f-g)(-1) = 0$
\item  $(g-f)(1) = -8$
\item  $(fg)\left(\frac{1}{2}\right) = -\frac{17}{16}$
\item  $\left(\frac{f}{g}\right)(0) = 0$
\item  $\left(\frac{g}{f}\right)\left(-2\right) = \frac{3}{16}$
\end{itemize}
\end{multicols}
}
