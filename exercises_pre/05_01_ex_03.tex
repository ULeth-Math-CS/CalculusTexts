{$f(x) = 4-3x$, $g(x) = |x|$}
{For   $f(x) = 4-3x$ and  $g(x) = |x|$,
\begin{multicols}{2}

\begin{itemize}

\item  $(g\circ f)(0) = 4$

\item  $(f\circ g)(-1) = 1$

\item  $(f \circ f)(2) = 10$

\item  $(g\circ f)(-3) = 13$

\item  $(f\circ g)\left(\frac{1}{2}\right) = \frac{5}{2}$

\item  $(f \circ f)(-2) = -26$

\end{itemize}

\end{multicols}}
