{$f(x) = \sqrt{3-x}$, $g(x) = x^2+1$}
{For $f(x) = \sqrt{3-x}$ and $g(x) = x^2+1$,
\begin{multicols}{2}

\begin{itemize}

\item  $(g\circ f)(0) = 4$

\item  $(f\circ g)(-1) = 1$

\item  $(f \circ f)(2) = \sqrt{2}$

\item  $(g\circ f)(-3) = 7$

\item  $(f\circ g)\left(\frac{1}{2}\right) = \frac{\sqrt{7}}{2}$


\item  $(f \circ f)(-2) = \sqrt{3 - \sqrt{5}}$

\end{itemize}

\end{multicols}
}
