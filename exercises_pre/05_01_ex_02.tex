{$f(x) = 4-x$, $g(x) = 1-x^2$}
{For   $f(x) = 4-x$ and $g(x) = 1-x^2$,
\begin{multicols}{2}

\begin{itemize}

\item  $(g\circ f)(0) = -15$

\item  $(f\circ g)(-1) = 4$

\item  $(f \circ f)(2) = 2$

\item  $(g\circ f)(-3) = -48$

\item  $(f\circ g)\left(\frac{1}{2}\right) = \frac{13}{4}$

\item  $(f \circ f)(-2) = -2$

\end{itemize}

\end{multicols}}
