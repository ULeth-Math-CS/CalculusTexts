{$f(x) = x^2$ and $g(x) = 3x-1$}
{For $f(x) = x^2$ and $g(x) = 3x-1$

\begin{itemize}
\item $(f+g)(x) = x^2+3x-1$
      Domain: $(-\infty, \infty)$
\item $(f-g)(x) = x^2-3x+1$ 
      Domain:  $(-\infty, \infty)$
\item $(fg)(x) = 3x^3-x^2$
      Domain: $(-\infty, \infty)$
\item $\left(\frac{f}{g}\right)(x) = \frac{x^2}{3x-1}$
      Domain:  $\left(-\infty, \frac{1}{3} \right) \cup \left(\frac{1}{3}, \infty \right)$
\end{itemize}
}
