{$f(x) = 6-x-x^2$, $g(x) = x\sqrt{x+10}$}
{For  $f(x) = 6-x-x^2$ and $g(x) = x\sqrt{x+10}$,
\begin{multicols}{2}

\begin{itemize}

\item  $(g\circ f)(0) = 24$

\item  $(f\circ g)(-1) = 0$

\item  $(f \circ f)(2) = 6$

\item  $(g\circ f)(-3) = 0$

\item  $(f\circ g)\left(\frac{1}{2}\right) = \frac{27-2\sqrt{42}}{8}$

\item  $(f \circ f)(-2) = -14$

\end{itemize}

\end{multicols}}
