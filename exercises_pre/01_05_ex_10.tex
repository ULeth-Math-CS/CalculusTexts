{$f(x) = x^2+1$ and $g(x) = \dfrac{1}{x^2+1}$}
{For  $f(x) = x^2+1$ and $g(x) = \dfrac{1}{x^2+1}$

\begin{multicols}{2}
\begin{itemize}
\item  $(f+g)(2) =\frac{26}{5}$
\item  $(f-g)(-1) = \frac{3}{2}$
\item  $(g-f)(1) = -\frac{3}{2}$
\item  $(fg)\left(\frac{1}{2}\right) =1$
\item  $\left(\frac{f}{g}\right)(0) = 1$
\item  $\left(\frac{g}{f}\right)\left(-2\right) = \frac{1}{25}$
\end{itemize}
\end{multicols}
}
