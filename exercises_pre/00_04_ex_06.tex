{$z = -5+i$, $w = 4+2i$}
{For  $z = -5+i$ and  $w = 4+2i$

\begin{itemize}

\item $z+w = -1+3i$

\item $zw = -22-6i$

\item $z^2 = 24-10i$

\item $\frac{1}{z} = -\frac{5}{26} - \frac{1}{26} \,i$

\item $\frac{z}{w} = -\frac{9}{10} + \frac{7}{10} \, i$

\item $\frac{w}{z} = -\frac{9}{13} - \frac{7}{13} \,i$

\item $\overline{z} = -5-i$

\item $z\overline{z} = 26$

\item $(\overline{z})^2 = 24+10i$

\end{itemize}
}
