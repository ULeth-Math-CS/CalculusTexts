{$f(x) = x^2-4$ and $g(x) = 3x+6$}
{For $f(x) = x^2-4$ and $g(x) = 3x+6$

\begin{itemize}
\item $(f+g)(x) = x^2+3x+2$
      Domain: $(-\infty, \infty)$
\item $(f-g)(x) = x^2-3x-10$
      Domain:  $(-\infty, \infty)$
\item $(fg)(x) = 3x^3+6x^2-12x-24$
      Domain: $(-\infty, \infty)$
\item $\left(\frac{f}{g}\right)(x) = \frac{x-2}{3}$
      Domain:  $\left(-\infty, -2 \right) \cup \left(-2, \infty \right)$
\end{itemize}
}
