{$f(x) = \dfrac{x}{2}$ and $g(x) = \dfrac{2}{x}$}
{For  $f(x) = \frac{x}{2}$ and $g(x) = \frac{2}{x}$

\begin{itemize}
\item $(f+g)(x) = \frac{x^2+4}{2x}$
      Domain: $(-\infty, 0) \cup (0, \infty)$
\item $(f-g)(x) = \frac{x^2-4}{2x}$
      Domain:  $(-\infty,0) \cup (0, \infty)$
\item $(fg)(x) = 1$
      Domain: $(-\infty,0) \cup (0, \infty)$
\item $\left(\frac{f}{g}\right)(x) = \frac{x^2}{4}$
      Domain: $(-\infty,0) \cup (0, \infty)$
\end{itemize}
}
