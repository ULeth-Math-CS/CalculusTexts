{$f(x) = 1-4x$ and $g(x) = 2x-1$}
{For $f(x) = 1-4x$ and $g(x) = 2x-1$

\begin{itemize}
\item $(f+g)(x) = -2x$
      Domain: $(-\infty, \infty)$
\item $(f-g)(x) = 2-6x$
      Domain:  $(-\infty, \infty)$
\item $(fg)(x) = -8x^2+6x-1$ 
      Domain: $(-\infty, \infty)$
\item $\left(\frac{f}{g}\right)(x) = \frac{1-4x}{2x-1}$ 
      Domain:  $\left(-\infty, \frac{1}{2} \right) \cup \left(\frac{1}{2}, \infty \right)$
\end{itemize}
}
