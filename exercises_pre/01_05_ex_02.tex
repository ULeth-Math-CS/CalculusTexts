{$f(x) = x^2$ and $g(x) = -2x+1$}
{For  $f(x) = x^2$ and $g(x) = -2x+1$

\begin{multicols}{2}
\begin{itemize}
\item  $(f+g)(2) = 1$
\item  $(f-g)(-1) = -2$
\item  $(g-f)(1) = -2$
\item  $(fg)\left(\frac{1}{2}\right) = 0$
\item  $\left(\frac{f}{g}\right)(0) = 0$
\item  $\left(\frac{g}{f}\right)\left(-2\right) = \frac{5}{4}$

\end{itemize}
\end{multicols}
}
