{$f(x) = 2x-5$}
{For $f(x) = 2x-5$

\begin{itemize}
\item  $f(2) = -1$
\item  $f(-2) = -9$
\item  $f(2a) = 4a-5$
\item  $2 f(a) = 4a-10$
\item $f(a+2) = 2a-1$
\item $f(a) + f(2) = 2a-6$
\item  $f \left( \frac{2}{a} \right) = \frac{4}{a} - 5 = \frac{4-5a}{a}$
\item $\frac{f(a)}{2} =\frac{2a-5}{2}$
\item  $f(a + h) = 2a + 2h - 5$
\end{itemize}
}
