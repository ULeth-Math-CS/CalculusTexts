{$f(x) = x^2-x$ and $g(x) = 7x$}
{For $f(x) = x^2-x$ and $g(x) = 7x$

\begin{itemize}
\item $(f+g)(x) = x^2+6x$
      Domain: $(-\infty, \infty)$
\item $(f-g)(x) = x^2-8x$
      Domain:  $(-\infty, \infty)$
\item $(fg)(x) = 7x^3-7x^2$
      Domain: $(-\infty, \infty)$
\item $\left(\frac{f}{g}\right)(x) = \frac{x-1}{7}$
      Domain:  $\left(-\infty, 0 \right) \cup \left(0, \infty \right)$
\end{itemize}
}
