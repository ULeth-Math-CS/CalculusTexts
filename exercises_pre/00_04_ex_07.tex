{$z = \sqrt{2} - i\sqrt{2}$, $w = \sqrt{2} + i\sqrt{2}$}
{For  $z = \sqrt{2} - i\sqrt{2}$ and $w = \sqrt{2} + i\sqrt{2}$

\begin{itemize}

\item $z+w = 2\sqrt{2}$

\item $zw = 4$

\item $z^2 = -4i$

\item $\frac{1}{z} = \frac{\sqrt{2}}{4} + \frac{\sqrt{2}}{4} \,i$

\item $\frac{z}{w} = -i$

\item $\frac{w}{z} = i$

\item $\overline{z} = \sqrt{2}+i\sqrt{2}$

\item $z\overline{z} = 4$

\item $(\overline{z})^2 = 4i$

\end{itemize}
}
