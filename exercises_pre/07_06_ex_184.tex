 {$\sin \left( \dfrac{1}{2}\arctan(x) \right)$ }
{ $\sin \left( \dfrac{1}{2}\arctan(x) \right) = \left\{ \begin{array}{rr} \sqrt{\dfrac{\sqrt{x^{2} + 1} - 1}{2\sqrt{x^{2} + 1}}} & \text{for $x \geq 0$} \\ [10pt] -\sqrt{\dfrac{\sqrt{x^{2} + 1} - 1}{2\sqrt{x^{2} + 1}}} & \text{for $x < 0$}  \end{array}\right. $ }
