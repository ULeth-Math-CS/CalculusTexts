{$f(x) = x^2$ and  $g(x) = \dfrac{1}{x^2}$}
{For  $f(x) = x^2$ and $g(x) = \dfrac{1}{x^2}$

\begin{multicols}{2}
\begin{itemize}
\item  $(f+g)(2) =\frac{17}{4}$
\item  $(f-g)(-1) = 0$
\item  $(g-f)(1) = 0$
\item  $(fg)\left(\frac{1}{2}\right) =1$
\item  $\left(\frac{f}{g}\right)(0)$ is undefined.
\item  $\left(\frac{g}{f}\right)\left(-2\right) = \frac{1}{16}$
\end{itemize}
\end{multicols}
}
