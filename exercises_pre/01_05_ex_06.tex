{$f(x) = \sqrt{4-x}$ and $g(x) = \sqrt{x+2}$}
{For $f(x) = \sqrt{4-x}$ and $g(x) = \sqrt{x+2}$

\begin{multicols}{2}
\begin{itemize}
\item  $(f+g)(2) = 2+\sqrt{2}$
\item  $(f-g)(-1) = -1+\sqrt{5}$
\item  $(g-f)(1) = 0$
\item  $(fg)\left(\frac{1}{2}\right) = \frac{\sqrt{35}}{2}$
\item  $\left(\frac{f}{g}\right)(0) = \sqrt{2}$
\item  $\left(\frac{g}{f}\right)\left(-2\right) = 0$
\end{itemize}
\end{multicols}
}
