{$f(x) = x^2$ and $g(x) = \dfrac{3}{2x-3}$}
{For  $f(x) = x^2$ and $g(x) = \dfrac{3}{2x-3}$

\begin{multicols}{2}
\begin{itemize}
\item  $(f+g)(2) = 7$
\item  $(f-g)(-1) = \frac{8}{5}$
\item  $(g-f)(1) = -4$
\item  $(fg)\left(\frac{1}{2}\right) = -\frac{3}{8}$
\item  $\left(\frac{f}{g}\right)(0) = 0$
\item  $\left(\frac{g}{f}\right)\left(-2\right) = -\frac{3}{28}$
\end{itemize}
\end{multicols}
}
