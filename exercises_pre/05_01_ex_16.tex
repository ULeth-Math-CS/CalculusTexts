{$f(x) = 3x-5$, $g(x) = \sqrt{x}$ }
{For   $f(x) = 3x-5$ and $g(x) = \sqrt{x}$ 

\begin{itemize}

\item  $(g \circ f)(x) = \sqrt{3x-5}$, domain: $\left[ \frac{5}{3}, \infty \right)$

\item  $(f \circ g)(x) = 3\sqrt{x}-5$, domain: $[0,\infty)$

\item  $(f \circ f)(x) = 9x-20$, domain: $(-\infty, \infty)$

\end{itemize}}
