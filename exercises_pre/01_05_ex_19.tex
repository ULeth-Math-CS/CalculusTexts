{$f(x) = x$ and $g(x) = \sqrt{x+1}$}
{For   $f(x) =x$ and $g(x) = \sqrt{x+1}$

\begin{itemize}

\item $(f+g)(x) = x+\sqrt{x+1}$
      Domain: $[-1,\infty)$
\item $(f-g)(x) = x-\sqrt{x+1}$
       Domain: $[-1,\infty)$
\item $(fg)(x) = x\sqrt{x+1}$
       Domain: $[-1,\infty)$
\item $\left(\frac{f}{g}\right)(x) =\frac{x}{\sqrt{x+1}}$ 
       Domain: $(-1,\infty)$
\end{itemize}
}
